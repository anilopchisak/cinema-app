# 🎬 CinemaКино — Поиск и просмотр фильмов

**CinemaКино** — современный веб-сервис для поиска, фильтрации и просмотра фильмов.

📦 **Версия на чистом React (Strapi)**:
[github.com/anilopchisak/cinema-strapi](https://github.com/anilopchisak/cinema-strapi)

---

## ✨ Возможности

- 📚 **Интернационализация** - английский и русский язык интерфейса.
- 🔍 **Просмотр и поиск фильмов** - удобный каталог с бесконечной подгрузкой и кнопкой Наверх.
- 🎞️ **Детальная страница фильма** - информация, описание, трейлер.
- ▶️ **Просмотр трейлера** в модальном окне прямо на сайте, а также кнопка "Не хочу выбирать" со случайным фильмом в модальном окне.
- 🎯 **Фильтрация и сортировка** - по жанрам, году выпуска, рейтингу, алфавиту с debounce.
- 👤 **Авторизация и регистрация** - создайте аккаунт, чтобы добавлять фильмы в избранное.
- ⭐ **Избранное** - сохраняйте понравившиеся фильмы и просматривайте их в отдельном разделе.
- 📱 **Адаптивный дизайн** - удобно на десктопе, планшете и мобильном телефоне.
- 🎨 **Плавные анимации** - при переходах и появлении элементов (framer-motion, react-responsive).

А также:
- **SEO** - для улучшенной индексации страниц браузерами.
- **Lazy-loading** - для оптимизации загрузки приложения.
- **Скелетоны** - для более приятного и предсказуемого пользовательского опыта.

- **Язык**: TypeScript 5.x
- **Библиотека**: React 19
- **Фреймворк**: Next.js 16.1.6 (App Router, серверные компоненты)

  - [TanStack Query (React Query) v5](https://tanstack.com/query/latest) — серверное состояние,
    кэширование, бесконечная подгрузка

- **Формы и валидация**:
  - [React Hook Form 7](https://react-hook-form.com/)
  - [Zod 4](https://zod.dev/) — схемы валидации
- **Стилизация**:
  - [Sass (SCSS)](https://sass-lang.com/) — модульные стили
  - [classnames](https://github.com/JedWatson/classnames) — условное объединение классов
- **Анимации**:
  - [Framer Motion 12](https://www.framer.com/motion/)
- **UI-кит**:
  - [react-icons](https://react-icons.github.io/react-icons/) — иконки
  - [react-toastify](https://fkhadra.github.io/react-toastify/) — уведомления
- **Утилиты**:
  - [lodash](https://lodash.com/) — debounce и другие полезные функции
  - [qs](https://github.com/ljharb/qs) — парсинг query-строк
  - [js-cookie](https://github.com/js-cookie/js-cookie) — работа с куками
- **Адаптивность**:
  - [react-responsive](https://github.com/contra/react-responsive) — медиа-запросы в коде
- **Интернационализация**
  - i18next

- **Линтер**: ESLint 9 (плоская конфигурация) + `eslint-config-next`

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**

   ```bash
   git clone git@github.com:anilopchisak/cinema-app.git
   cd cinema-app
   ```

2. **Установите зависимости** (рекомендуется `npm ci` для чистой установки)

   ```bash
   npm ci
   ```

3. **Настройте переменные окружения**  
   Создайте файл `.env` в корне проекта и укажите базовый URL вашего Strapi-бэкенда:

   ```env
   NEXT_PUBLIC_STRAPI_BASE_URL=url
   ```

4. **Запустите проект в режиме разработки**

   ```bash
   npm run dev
   ```

   Откройте [http://localhost:3000](http://localhost:3000) в браузере.

5. **Сборка для продакшена**
   ```bash
   npm run build
   npm start
   ```
